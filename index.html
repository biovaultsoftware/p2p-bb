This demo is fully offline. It appends signed STAs locally and replays deterministically.
For peer delivery (WebRTC + signal fabric), wire your servers later — this is the stable local core.
Rules
• State is authoritative only after appendSTA() succeeds.
• Append is atomic: chain + nonce log + interpreted store in one transaction.
• Determinism: deep canonical JSON + SHA-256.
• Replay attacks rejected by nonce reuse.

Install on iPhone/iPad

iOS Safari does not show an Install button. Use Share → Add to Home Screen.

This app keeps data locally. If your iOS version isolates storage between Safari and the Home Screen app, use Export before installing and Import after opening from the Home Screen.
